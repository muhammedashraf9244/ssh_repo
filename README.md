
# **Foobar** 
 Foobar is a python library for dealing world 

## **Installation** 

use package manager [pip](https://pypi.org/project/django-foobar/) to install foobar 

    pip install foobar 

## **Usage** 
```
import foo.bar
foobar.pluralize('word')
foobar.pluralize('goose')
foobar.pluralize('phonemone')
```
